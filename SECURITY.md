# Security Policy

## Supported Versions

The following versions currently receive security updates.

| Version | Supported |
|----------|-----------|
| 0.2.x    | ✅ |
| 0.1.x    | ❌ |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public GitHub issue.

Instead, contact the maintainer privately and include:

- Home Assistant version
- LG ESS firmware version
- Integration version
- Description of the issue
- Steps to reproduce
- Potential impact

Security reports will be acknowledged as soon as possible.

---

## Scope

Examples include:

- Unauthorized access to the ESS
- Authentication or authorization issues
- Sensitive information exposure
- Remote code execution
- Denial of service vulnerabilities

General bugs or feature requests should be reported through the GitHub issue tracker.

---

## Disclosure Policy

Security vulnerabilities will be investigated privately.

Once a fix is available, the issue will be disclosed responsibly together with the corresponding release notes.

---

Thank you for helping keep **LG ESS Local** secure.
